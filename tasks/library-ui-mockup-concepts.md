## Animation & Transition Notes

### Card Entrance Animation
```
Initial:  opacity: 0, translateY: 20px
Final:    opacity: 1, translateY: 0
Duration: 200ms
Easing:   ease-out
Stagger:  50ms delay per card
```

### Status Badge Pulse (URGENT)
```
Keyframes:
0%:   scale: 1, opacity: 1
50%:  scale: 1.05, opacity: 0.8
100%: scale: 1, opacity: 1
Duration: 2s
Timing:   ease-in-out
Repeat:   infinite
```

### Progress Bar Fill
```
Initial:  width: 0%
Final:    width: 85% (or actual value)
Duration: 800ms
Easing:   ease-out
```

### Swipe Gesture
```
Threshold: 50px swipe distance
Snap:      elastic spring animation
Duration:  300ms
```

### Modal Entrance (Bottom Sheet)
```
Initial:  translateY: 100%
Final:    translateY: 0
Duration: 300ms
Easing:   cubic-bezier(0.4, 0, 0.2, 1)
```
